Abstract

PYTHERA Detection is a hybrid object detection model that combines CNN backbone and Transformer architecture, designed to address the inference speed issues of traditional detection models like YOLO.

Problem

Traditional detection models like YOLO have an NMS (Non-Maximum Suppression) layer that makes inference slow, making them unsuitable for real-time applications.

Solution

Combining a hybrid architecture between YOLO and DETR, using the query method in transformer to eliminate the NMS layer, maintaining performance while significantly improving inference speed.

Traditional YOLO vs DETR Comparison

Traditional YOLO must go through the NMS layer, while DETR only requires queries, resulting in significantly improved speed.

Dataset

COCO Dataset

Used for training and initial performance evaluation of the model through the mAP50 metric. This dataset is the industry standard for object detection.

Human Crowded Dataset

A custom-built dataset to evaluate performance on the human detection problem. This dataset has the following characteristics:

  • Bounding boxes with significant overlap
  • Bounding boxes misaligned from the original objects
  • Goal: evaluate performance for human detection tasks

Model Architecture

The PYTHERA Detection architecture combines the CNN backbone from YOLOv9 with the transformer decoder from DETR. This allows the model to:

  • Maintain the high performance of YOLOv9
  • Eliminate the NMS layer through the query mechanism in transformer
  • Achieve significantly faster inference speed

Results

1. Results on COCO Dataset

After training on the COCO dataset, PYTHERA model only lags behind YOLOv9 by approximately 2% in mAP50, demonstrating that performance is well maintained.

2. Results on Human Crowded Dataset

Comparing PYTHERA Detection with YOLOv9, YOLO-DETR, and YOLOS on the Human Crowded dataset:

Results: PYTHERA Detection (YOLOv9 + DETR) completely outperforms YOLOS and DETR. Compared to YOLOv9, it only lags behind by approximately ~1%.
